Mosaic

A web application for the Institute of Personality and Social Research

Mosaic is a web application my team developed for the Institute of Personality and Social Research. It is an online version of a real-world test used by psychologists to assess aesthetic predispositions. Technologies used to build the application include Ruby on Rails, Javascript, and Heroku.

  • The mosaic construction page where participants create their mosaic via drag-and-drop actions.

  • Upon finishing, participants submit their mosaics.

  • Researchers are able to view the step-by-step construction of the mosaic by using the slider on the mosaic view page.

  • The gallery page allows researchers to filter completed mosaics by a variety of traits such as "Total number of colors used", "Total number of moves", and "Dominant color used". It also provides researchers with the ability to download the mosaic data in the form of .csv files for analysis and interpretation.
